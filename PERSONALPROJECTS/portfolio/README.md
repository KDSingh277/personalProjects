# Portfolio (Karandeep Singh)

Static portfolio site built from the resume you provided.

## Preview locally

- Open `index.html` directly in your browser, or use a simple local server:

```powershell
cd PERSONALPROJECTS\portfolio
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## Deploy (GitHub Pages)

This project is designed to be deployed as a static site (no build step).
